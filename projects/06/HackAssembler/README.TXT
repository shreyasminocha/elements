PROJECT TITLE: HackAssembler
PURPOSE OF PROJECT: This is an assembler for converting the Hack 
assembly language to the Hack machine code as described on 
www.nand2teris.org/06.php.

VERSION or DATE: 26 September, 2016
HOW TO START THIS PROJECT: Open the package file in BlueJ.
USER INSTRUCTIONS: Create an instance of Main, 'm' passing the path of the 
file to be assembled as an argument to the method. Run m.main(). 
The assembled file will be in the same directory as the original file, 
with the extension '.hack'.
