PROJECT TITLE: VMTranslator
PURPOSE OF PROJECT: To convert a vm file, or a directory of vm files to 
a single assembly language program as described on 
www.nand2tetris.org/07.php and www.nand2tetris.org/08.php.

VERSION or DATE: 19 October, 2016
HOW TO START THIS PROJECT: Open the project in BlueJ.
USER INSTRUCTIONS: Create an instance of Main, 'm' passing the path of the
file or directory to be converted as an argument to the method.Run m.main(). 
The '.asm' file will be in the same directory as the '.vm' file(s) that was
converted.
